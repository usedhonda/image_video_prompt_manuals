# Grok Imagine Camera Movement Keywords

## Overview

Camera movements significantly impact the cinematic quality of generated videos. Grok Imagine responds well to cinematography terminology.

---

## Basic Camera Movements

| Keyword | Effect | Example Usage |
|---------|--------|---------------|
| `pan left/right` | Horizontal rotation | `camera pans right to reveal cityscape` |
| `tilt up/down` | Vertical rotation | `tilt up from feet to face` |
| `dolly in/out` | Forward/backward movement | `slow dolly in on subject's face` |
| `zoom in/out` | Lens zoom | `zoom out to reveal full scene` |
| `tracking shot` | Follow subject movement | `tracking shot following runner` |
| `orbit` | Circle around subject | `orbiting camera around product` |

---

## Shot Types

| Keyword | Description |
|---------|-------------|
| `extreme close-up` | Tight on detail (eyes, hands) |
| `close-up` | Face or object fills frame |
| `medium close-up` | Head and shoulders |
| `medium shot` | Waist up |
| `medium full shot` | Knees up |
| `full shot` | Entire body |
| `wide shot` | Subject with environment |
| `extreme wide shot` | Landscape with small subject |
| `establishing shot` | Sets scene location |

## Grok-Specific Keywords

| Keyword | Effect |
|---------|--------|
| `unfixed lens` | Enables camera movement |
| `Shot Switch` | Cut to new scene/angle |
| `POV` | First-person perspective |
| `pull-back` | Zoom out revealing more |

## Combination Examples

### Cinematic Reveal
```
Slow crane up from street level, revealing vast cityscape at golden hour
```

### Product Hero
```
Medium zoom pull-back with orbiting camera, revealing product from all angles
```

### Action Sequence
```
Handheld tracking shot following runner through urban alley, POV moments
```

### Documentary Style
```
Steadicam medium shot following subject through busy marketplace
```

---

## Multi-Shot Sequences

Use `Shot Switch.` to combine different camera setups:

```
Wide establishing shot of mountain landscape. Shot Switch.
Close-up of hiker's boots on rocky trail. Shot Switch.
POV looking up at mountain peak.
```

**Note**: Ensure `unfixed lens` mode is enabled for shot switches.

---

## Best Practices

1. **One primary movement** per shot - avoid combining pan + zoom + dolly
2. **Match movement to mood** - slow for emotional, fast for energetic
3. **Front-load camera direction** - mention movement early in prompt
4. **Use lens terminology** for specific looks
5. **Specify speed** when precision matters
