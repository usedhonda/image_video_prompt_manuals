## 1. 7つのプロンプトコツ

### コツ①: 情景が目に浮かぶように具体的に描写

詳細な情報を加えるほど、イメージを正確に捉えられる。

| Before | After |
|--------|-------|
| おしゃれなカフェの店内を生成して | 雨が降る窓の外を眺められる、静かな路地裏のカフェを生成して。木製のカウンターにはコーヒーサイフォンが置かれ、温かみのあるオレンジ色の照明が店内を照らしている。読書をしている人が一人だけいる状態にして |

### コツ②: キャラクターの個性を保ち続ける

同じキャラクターで繰り返し生成するうちに姿が変わる場合、新しい会話を始めて特徴を再指定。

```
以下の探偵キャラクターを、様々なシチュエーションで描いて！
髪型: 少し癖のある黒髪を無造作に分けている
表情: 鋭い観察眼を持つ、思慮深い表情
服装: 着古したトレンチコートと、緩く締められたネクタイ
小道具: いつも手元に虫眼鏡か、メモ帳を持っている
スタイル：可愛らしいクレイアニメ風
まずは、この探偵が深夜のオフィスで証拠を探している場面を生成して！
```

### コツ③: 画像の「目的」や「背景」を伝える

| Before | After |
|--------|-------|
| ノートパソコンとコーヒーが置かれた机を生成して | リモートワークの生産性向上術というテーマのブログ記事に使うヘッダー画像を作成したい。朝日が差し込む明るい書斎で、ノートパソコンの隣にコーヒーカップと観葉植物が置かれている、爽やかでポジティブな印象の写真を生成して |

### コツ④: 対話を通じて少しずつ理想へ

一度で完璧を目指さず、対話で修正を重ねる。

```
1回目: 魔法のポーションが入った、ガラスの小瓶を生成して
2回目: いいですね。瓶の形を、もっと丸みを帯びたフラスコのような形にできますか？中の液体は、星屑のようにキラキラと輝かせてほしい。
3回目: 素晴らしいです！最後に、瓶のコルク栓に古い羊皮紙のタグが結び付けられている様子を追加してください。
```

### コツ⑤: 「〜がない」ではなく「〜がある」で伝える

否定形より肯定形で望む光景を描写。

| Before | After |
|--------|-------|
| 人工的な建物がなく、騒音もない、静かな湖畔を生成して | 風のない穏やかな朝、鏡のように空を映し出す静かな湖畔、聞こえるのは鳥のさえずりと、時折聞こえる水の音だけ。ありのままの自然が広がっている様子を生成して |
| 肉や油っこいドレッシングを使っていない、健康的なサラダ | 採れたての新鮮なベビーリーフ、真っ赤なミニトマト、瑞々しいキュウリを使った彩り豊かなサラダ。オリーブオイルと岩塩だけで仕上げた、素材の味を活かした一皿 |

### コツ⑥: アスペクト比を調整する

プロンプトで直接指定可能。

```
スマートフォンの壁紙用に、夜空に浮かぶ幻想的なオーロラの画像を、アスペクト比 9:16 で生成して
```

**対応アスペクト比**: `1:1`, `2:3`, `3:2`, `3:4`, `4:3`, `4:5`, `5:4`, `9:16`, `16:9`, `21:9`

### コツ⑦: カメラワークを指示して構図を操る

写真・映画の専門用語で構図をコントロール。

| カテゴリ | キーワード例 |
|----------|-------------|
| 撮影範囲 | 広角ショット、マクロショット（接写） |
| カメラ位置 | ローアングル（低い位置からのあおり）、俯瞰、鳥瞰図（バードアイビュー） |
| レンズ種類 | 85mm のポートレートレンズ |
| 独特な表現 | ダッチアングル（傾いた構図） |

**プロンプト例**:
```
空飛ぶ車が行き交う未来の都市を、超高層ビルの屋上から見下ろした、鳥瞰図（バードアイビュー）で生成して
```

## 2. Nano Banana Pro専用機能

Nano Banana Pro（gemini-3-pro-image-preview）でのみ使用可能な高度機能。

### 思考プロセス（Thinking）

AIが描く前に「考える」。`include_thoughts=True`で思考を覗ける。

**プロンプト例**:
```
Create an unusual but realistic image that might go viral
（バイラルしそうな、変わっているがリアルな画像を作成）
```

### 検索グラウンディング（Search Grounding）

Google検索からリアルタイムデータを取得して描画。

**プロンプト例**:
```
Visualize the current weather forecast for the next 5 days in Tokyo as a clean, modern weather chart in Japanese. add a visual on what i should wear each day
（今後5日間の東京の天気予報を日本語でモダンな天気図として視覚化。各日に何を着るべきかのビジュアルも追加）
```

### 4K生成

高解像度出力。`image_size="4K"`で指定。

**プロンプト例**:
```
A photo of an oak tree experiencing every season
（すべての季節を経験しているオークの木の写真）
```

### 多言語バナナ（Polyglot Banana）

画像内テキストの生成・翻訳（12言語以上対応）。

```
1. Make an infographic explaining Einstein's theory of General Relativity suitable for a 6th grader in Spanish
（アインシュタインの一般相対性理論のインフォグラフィックをスペイン語で）

2. Translate this infographic in Japanese, keeping everything else the same
（このインフォグラフィックを日本語に翻訳、他はそのまま）
```

### 高度な画像ミキシング

最大**14枚**の画像をミックス可能（Flashは3枚まで）。

**プロンプト例**:
```
An office group photo of these people, they are making funny faces.
（これらの人々のオフィス集合写真、変顔をしている）
```

**Tips**: キャラクターの忠実度を高く保つなら5枚程度がベスト。

## 3. ベストプラクティス8箇条

| # | ルール | 詳細 |
|---|--------|------|
| 1 | 超具体的に | 色、光、構図を細かく指定 |
| 2 | 文脈を伝える | 「何のために」「どんなムードで」 |

| 5 | ポジティブに | 否定形より肯定形 |

| 7 | 検索を活用 | リアルデータが欲しいときは「Webで検索して」 |
| 8 | Batch APIで節約 | 急がないなら半額（最大24時間待ち） |

## 4. 動画生成連携テクニック

Nano Banana → 動画生成AI（Veo/Kling等）のワークフロー。

### 基本フロー

```
[Nano Banana Pro]    →    [動画生成AI]    →    [編集]
   画像生成               動画化              仕上げ
   ・構図変更             ・Veo 3.1
   ・商品差し替え         ・Kling 2.1
   ・人物生成             ・Seedance
```

### 動画フレーム用プロンプト例

**商品撮影（スムージー）**:
```
Close-up of a green smoothie swirling, vibrant colors, studio lighting, healthy ingredients.
```

**商品差し替え**:
```
この○○の画像を○○に変えて
```

**人物アクション（飲むシーン）**:
```
A woman drinking her beverage deliciously. The angle at which she tilts the glass is 30 degrees. Not too much.
（女性が美味しそうに飲み物を飲んでいる。グラスを傾ける角度は30度。ほどよく傾けている。）
```

**Tips**: 飲むシーンで角度を指定しないと液体の描写が不自然になる。

### 動画化プロンプト例

| シーン | プロンプト |
|--------|-----------|
| 攪拌 | `stirring` |
| カメラに向かって笑顔 | `smiling at camera` |
| 飲む動作 | 角度指定必須（上記参照） |

## 5. クリエイティブ活用例

### 塗り絵変換
```
この画像のモチーフのみを抽出して塗り絵にして。光と影を完全に無視して
```

### 間取り図→3Dレンダリング
```
この住宅の間取り図すべてを、俯瞰図のまま等角投影のフォトリアルな3Dレンダリングに変換して画像生成してください。
```

### ミニチュアフィギュア化
```
入力画像の被写体をリアルなミニチュア版として高解像度で撮影した広告写真。人物、キャラクター、生物を被写体とする場合、全身が写り、比例して縮小されていること。清潔な白背景、スタジオ照明、柔らかな影。手は手入れが行き届き、自然な肌色で、製品の形状と細部を強調する位置に配置。ミニチュア被写体は極小ながら超精密でブランドイメージを忠実に再現し、浅い被写界深度でフレーム中央に配置。高級品写真とミニマルな商業スタイルを模倣した表現
```

### スタンプセット化
```
添付した画像をモデルに、この子の毛色、模様、顔立ち、体型や骨格、全体のシルエットといった身体的特徴を忠実に再現した丸型のスタンプセットを作成してください。

■レイアウト:
・スタンプは 3x3 のグリッドに合計 9 個配置
・スタンプが押される背景は、日本の和紙のような質感のあるホワイト
・各丸型スタンプの下部の縁に沿って「Made with Gemini」

■内容:
・それぞれのスタンプは異なるポーズや表情
・顔のクローズアップ、上半身、全身の構図がバランス良く

■スタイルと質感:
・日本の伝統的な朱肉で押したようなリアルな質感
・色はすべて「朱肉」の赤一色、手で押したような濃淡やかすれ
```

### カロリー計算インフォグラフィック
```
アップロードされた写真内の料理を自動分析し、カロリー注釈付きのスタイリッシュなインフォグラフィック「分解図」を生成します。
# 手順
- まず、写真内の料理を特定します。
- 次に、その主要な構成要素を自動判定し、視覚的に分離して宙に浮いているように見せます。
- 分離した各成分に「名称」と「推定カロリー」を明瞭なフォントで注釈として追加。
- 全体の美観は高級料理雑誌の特集ページのように洗練されプロフェッショナルな印象に。
- 背景は純白または薄いグレーのミニマルスタイル。
- 左上隅に「Gemini Calorie Calculator」のテキストを配置。
```
