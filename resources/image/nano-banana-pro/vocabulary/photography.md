## Camera Settings

| Setting | Values | Effect |
|---------|--------|--------|
| Aperture | `f/1.4` - `f/22` | f/1.4 = shallow DOF, f/16 = deep focus |
| Shutter Speed | `1/2000s` - `30s` | Fast = freeze motion, Slow = blur |
| ISO | `100` - `12800` | Low = clean, High = grain |
| White Balance | `2700K` - `7500K` | 3200K = warm, 5600K = daylight, 7000K = cool |

---

## Visual Quality Artifacts (Realism)

Add intentional imperfections for photorealism:
- `subtle film grain` - Analog texture
- `chromatic aberration on edges` - Lens imperfection
- `slight motion blur` - Natural movement
- `lens vignette` - Edge darkening
- `sensor noise` - High ISO simulation
- `dust particles` - Atmospheric realism

## Composition

### Framing
| Term | Description | Best For |
|------|-------------|----------|
| `extreme close-up` | Face/detail only | Emotional emphasis |
| `close-up` | Head and shoulders | Dialogue scenes |
| `medium close-up` | Chest up | Standard portrait |
| `medium shot` | Waist up | Conversation |
| `medium full` | Knees up | Partial action |
| `full shot` | Entire body | Character reveal |
| `wide shot` | Body + environment | Scene establishment |
| `extreme wide` | Vast landscape | Epic scale |

### Camera Angle
| Term | Effect |
|------|--------|
| `eye level` | Neutral, relatable |
| `low angle` | Power, dominance |
| `high angle` | Vulnerability, overview |
| `bird's eye view` | God-like perspective |
| `worm's eye view` | Extreme power, awe |
| `dutch angle` | Unease, tension |
| `over the shoulder` | POV, dialogue |
| `45 degrees left/right` | Dynamic, interest |

### Composition Rules
- `rule of thirds` - Subject on intersection points
- `center frame` - Symmetrical emphasis
- `golden ratio` - Natural flow
- `diagonal` - Dynamic tension
- `leading lines` - Guide viewer's eye
- `frame within frame` - Depth and focus
