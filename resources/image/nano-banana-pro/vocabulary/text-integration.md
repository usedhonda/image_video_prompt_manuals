# Text Integration

Keywords for natural text embedding in images.

---

| Keyword | Effect | Use Case |
|---------|--------|----------|
| Kinetic Typography | Motion/speed-feeling text design | Action posters, dynamic ads |
| Diegetic Text | Text exists physically in scene | Neon signs, book covers, T-shirt logos |
| Embossed / Debossed | Raised or recessed text | Business cards, leather, stone tablets |
| Neon Signage | Gas-tube glow with ambient light | Cyberpunk streets, bar signs |
| Volumetric Text | Text as light beams in space | Concert visuals, mystical titles |
| Caustic Projection | Transparent text projects shadow letters | Perfume bottles, glass objects |
| Gold Foil Stamping | Metallic gold finish on paper | Invitations, luxury packaging |
| Voxel Art Text | 3D cube-built letters | Game UI, LEGO-style design |
| Written in [Material] | Text made of specified material | Cloud text, sauce letters |
| Knolling Layout | Text in organized right-angle arrangement | Gadget intros, desk setup |
| Calligraphy / Ink Wash | Brush strokes with ink bleeding | Japanese menus, martial arts |
| Retro-Reflective | Bright reflective material like road signs | Street fashion, construction |
