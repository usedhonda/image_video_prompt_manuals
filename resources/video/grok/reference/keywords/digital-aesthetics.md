# Digital Aesthetics

Internet-native visual style keywords.

---

| Keyword | Effect | Use Case |
|---------|--------|----------|
| Glitch Art / Datamosh | Data corruption, compression noise | Cyberpunk, hacking, unsettling |
| Vaporwave | 80-90s nostalgia, pastels, ancient statues | Retrofuture, memes, chill vibes |
| CRT Scanlines | Tube TV lines, RGB pixel shift | Retro games, surveillance, VHS |
| Bit-crushed / 8-bit | Low resolution, limited colors | Retro games, lo-fi aesthetic |
| Synthwave / Outrun | Neon pink/blue grid, sunset, sports cars | 80s sci-fi, night drive |
| Low Poly | Blocky, low polygon count 3D | PS1 style, minimal art, indie games |
| Dithering | Dot patterns for limited-color gradients | PC-98 graphics, GIF, pixel art |
| Chromatic Aberration | Color fringing (red/blue edge shift) | Camera imperfection, psychedelic |
| Halftone Pattern | Dot-based shading like comics/newspapers | Pop art, manga panels, posters |
| ASCII Art | Images drawn with text characters | Terminal screens, Matrix, hacker |
| Frutiger Aero | 2000s glossy, transparent, nature+tech | Windows Vista, optimistic future |
| Liminal Space | Empty, eerie, nostalgic artificial spaces | Dreams, backrooms, abandoned malls |
